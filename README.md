# Algonive_AI_Chatbot
Algonive Python Programming Internship Task Two
Project Description
This repository is dedicated to the second task of my Python Programming internship at Algonive. The objective is to build an AI Chatbot. This chatbot is designed to handle customer support queries and provide responses in real-time.
My goal for this project was to explore how automation can improve user engagement in sectors like banking and healthcare. I have kept the logic simple and functional to ensure ease of use.
Features
Understanding user queries using Natural Language Processing.
Providing predefined and generated responses for common questions.
Logic to handle various types of customer support inquiries.
How to Use
Clone the repository to your computer.
Make sure Python is installed on your device.
Run the application script: python chatbot.py
Type your questions in the terminal to interact with the bot.
